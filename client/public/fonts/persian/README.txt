# فونت‌های فارسی مورد نیاز

برای استفاده از فونت‌های فارسی در تله‌پرامپتر، لطفاً فایل‌های زیر را در این پوشه قرار دهید:

1. **ایران‌سنس (IRANSans)**:
   - IRANSansWeb.ttf
   - IRANSansWeb.woff
   - IRANSansWeb.woff2

2. **وزیر (Vazir)**:
   - Vazir.ttf
   - Vazir.woff
   - Vazir.woff2

3. **ساحل (Sahel)**:
   - Sahel.ttf
   - Sahel.woff
   - Sahel.woff2

می‌توانید این فونت‌ها را از منابع زیر دانلود کنید:

- [فونت ایران‌سنس](https://github.com/rastikerdar/vazirmatn)
- [فونت وزیر](https://github.com/rastikerdar/vazirmatn)
- [فونت ساحل](https://github.com/rastikerdar/sahel-font)

نکته: اگر فونت‌ها در پوشه local قرار داده نشوند، برنامه از CDN استفاده خواهد کرد که ممکن است به دلیل محدودیت‌های CORS با مشکل مواجه شود.
